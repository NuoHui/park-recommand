**5 大核心模块**:

2. **执行沙箱** (Execution Sandbox)
   - ToolRegistry 白名单验证
   - PreCheckExecutor 前置检查

3. **资源管理系统** (Resource Management)
   - RateLimiter - API 频率控制 (60/min)
   - TokenTracker - Token 追踪
   - ConcurrencyController - 并发控制 (5)
   - ResourceManager - 统一协调

4. **验证和风险系统** (Validation & Risk)
   - 意图验证 - 安全检查、权限验证
   - 风险评分 - 四维度评分 (工具/参数/深度/历史)
   - 风险等级 - Low/Medium/High/Critical

5. **监控和审计系统** (Monitoring & Audit)
   - SafetyMonitor - 5 种告警类型
   - ExecutionTracker - 完整审计追踪
