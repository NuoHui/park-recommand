# 首次运行

## 前置条件

确保你已完成 [安装指南](./installation.md) 中的所有步骤。

## 启动应用

### 开发模式

推荐首次运行使用开发模式，可以看到详细的日志输出：

```bash
npm run dev
```

应该看到以下输出：


如果要运行编译后的版本：



## 常见问题

### Q: 应用启动后无响应？

**A**: 
- 检查 API Key 配置是否正确（`.env` 文件）
- 查看日志输出是否有错误信息
- 检查网络连接是否正常

### Q: "API Key 无效" 错误？

**A**:
- 确认 API Key 已正确复制到 `.env` 文件
- 检查 LLM_PROVIDER 配置是否与 API Key 类型匹配
- 如使用 OpenAI，确保账号有有效的支付方式

### Q: 怎样退出应用？

**A**: 选择 "退出 (4)" 或按 `Ctrl+C`
