# LLM 测试流程图

## 快速测试流程

```
┌─────────────────────────────────────────────────────────────┐
│                  🧪 LLM 快速测试开始                         │
│                   npm run test:llm                          │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 1️⃣  环境配置验证                                            │
│ ├─ 检查 LLM_PROVIDER 设置                                    │
│ ├─ 检查 API_KEY 是否存在                                    │
│ └─ 检查模型名称配置                                        │
└─────────────────────────────────────────────────────────────┘
                            ↓
                      ✅ 通过 ❌ 失败
                      ↙          ↘
                    ✅            ❌
                    ↓              ↓
            继续测试          显示错误并停止
                    ↓
┌─────────────────────────────────────────────────────────────┐
│ 2️⃣  LLM 连接测试                                            │
│ ├─ 创建 LLM 客户端                                          │
│ ├─ 发送测试消息                                            │
│ └─ 验证响应                                                │
└─────────────────────────────────────────────────────────────┘
                            ↓
                      ✅ 通过 ❌ 失败
                      ↙          ↘
                    ✅            ❌
                    ↓              ↓
            继续其他测试      显示错误，跳过后续
                    ↓
┌─────────────────────────────────────────────────────────────┐
│ 3️⃣  基础调用测试                                            │
│ ├─ 调用 LLM: "请介绍深圳"                                  │
│ ├─ 验证响应内容                                            │
│ └─ 检查 Token 使用                                         │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 4️⃣  用户偏好提取测试                                        │
│ ├─ 输入: "我想在宝安找登山景点，距离不超过10公里"           │
│ ├─ 验证意图识别                                            │
│ └─ 验证信息提取                                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 5️⃣  推荐决策生成测试                                        │
│ ├─ 输入: 用户偏好对象                                      │
│ ├─ 生成搜索参数                                            │
│ └─ 验证推荐决策                                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│ 6️⃣  推荐解析测试                                            │
│ ├─ 输入: 景点列表 (JSON)                                   │
│ ├─ 解析结构化信息                                          │
│ └─ 验证推荐理由                                            │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│                    📊 生成测试报告                           │
│ ├─ 统计通过测试数                                          │
│ ├─ 统计失败测试数                                          │
│ └─ 计算成功率                                              │
└─────────────────────────────────────────────────────────────┘
                            ↓
                      ✅ 全部通过
                      ↙          ↘
                全部通过        部分失败
                  ↓               ↓
        ╔════════════════╗  ╔═══════════════════╗
        ║  🎉 完成      ║  ║  ⚠️  需要检查    ║
        ║  系统就绪     ║  ║  显示故障排除    ║
        ║  可进行推荐   ║  ║  建议              ║
        ╚════════════════╝  ╚═══════════════════╝
```

                             ↓

        ┌────────────────────────────────────────┐

        ├─ 测试 4: 引擎初始化

        │
