# 基础使用

## 启动应用

### 开发模式

```bash
npm run dev
```

热重载支持，适合开发调试。

### 生产模式

```bash
npm run build
npm start
```

编译优化后运行，适合实际使用。

## 基础流程

### 1. 启动推荐流程

启动应用后，自动进入推荐流程：

```
[i] 欢迎使用！我是你的深圳景点推荐助手。
```

### 2. 输入位置

```
[?] 请告诉我你的所在位置或地址：
> 南山区
```

支持以下位置格式：
- 行政区名称：`南山区`、`福田区`、`龙华区` 等
- 具体地点：`科技园`、`华强北`、`欢乐谷` 等
- 完整地址：`深圳市南山区科技园路 1 号`

### 3. 选择景点类型

```
[?] 你更喜欢哪种景点？
    > 公园 (P)
    > 爬山 (H)
    > 都可以 (B)
请选择 [P/H/B]: h
```

| 选项 | 说明 |
|------|------|
| `P` 或 `公园` | 城市公园、景点 |
| `H` 或 `爬山` | 山峰、登山路线 |
| `B` 或 `都可以` | 混合推荐 |

### 4. 选择距离范围

```
[?] 你希望景点距离多远（公里）？
    > 3 km 以内 (1)
    > 5 km 以内 (2)
    > 10 km 以内 (3)
    > 无限制 (4)
请选择 [1-4]: 2
```

| 选项 | 范围 |
|------|------|
| `1` | 3 km |
| `2` | 5 km |
| `3` | 10 km |
| `4` | 无限制 |

### 5. 查看推荐结果

```
[✓] 推荐完成！

┌─ 推荐结果 ─────────────────────────────┐
│ #1  梧桐山风景区                      │
│     距离: 3.2 km | 评分: ★★★★★       │
│     难度: 中等 | 时间: 2-3 小时        │
│                                     │
│ #2  翠竹山公园                       │
│     距离: 1.5 km | 评分: ★★★★☆       │
│     难度: 简单 | 时间: 1-2 小时        │
└─────────────────────────────────────┘
```

每个推荐项包含：
- **名称**: 景点名称
- **距离**: 与输入位置的距离
- **评分**: 用户评分（星级制）
- **难度**: 登山难度或景点评价
- **时间**: 预计游玩或登山时间

### 6. 后续操作

```
[?] 需要帮助吗？
    > 详情 (1)
    > 重新推荐 (2)
    > 路线规划 (3)
    > 退出 (4)
请选择 [1-4]: 1
```

#### 选择 1: 查看详情

```
[?] 请选择景点编号 [1-2]: 1

┌─ 梧桐山风景区 详情 ──────────────────┐
│ 地址: 深圳市罗湖区                   │
│ 距离: 3.2 km                       │
│ 评分: ★★★★★ (4.8/5)                │
│ 难度: 中等 (2/5)                    │
│ 预计时间: 2-3 小时                   │
│                                   │
│ 介绍:                              │
│ 梧桐山是深圳最高的山，山顶有        │
│ 观景台。登山路线完善。              │
│                                   │
│ 🗺️  导航: https://amap.com/...     │
│ 📞 咨询: 0755-8888-8888            │

#### 选择 2: 重新推荐

返回到第 2 步，可以修改位置或偏好重新生成推荐。

#### 选择 3: 路线规划

显示从当前位置到景点的路线信息：

```
[i] 正在规划路线...

┌─ 路线规划 ────────────────────────┐
│ 出发地: 南山区科技园              │

#### 选择 4: 退出

退出应用。

## 故障排查

### 无响应

- 检查网络连接
- 查看 [日志文件](../api/logging.md) 了解详细信息
- 参考 [常见问题](./troubleshooting.md)

### 位置识别错误

- 使用更具体的地址
- 使用行政区名称（如 "南山区" 而非 "南山"）

### 推荐结果不理想

- 尝试调整距离范围
- 尝试修改景点类型偏好
- 选择 "重新推荐" 获取其他选项
